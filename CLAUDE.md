# CLAUDE.md — Painel PILAR

Instruções permanentes para qualquer sessão de trabalho neste repositório.

O retrato técnico do sistema está em **`ARQUITETURA-PAINEL-PILAR.md`** (raiz) — leia-o
antes de mexer no código e mantenha-o atualizado (regra abaixo).

## MANUTENÇÃO DA DOCUMENTAÇÃO (OBRIGATÓRIO)

Sempre que você alterar o código do Painel PILAR (qualquer mudança em `painel.html`,
`painel-proxy.js`, `calc.js`, `calc-processo.js`, `index.html`, ou na estrutura de
dados/rotas), você DEVE atualizar o `ARQUITETURA-PAINEL-PILAR.md` na mesma sessão,
refletindo a mudança. Isso não é opcional — faz parte de toda tarefa de implementação.
Antes de reportar que terminou, confirme que o ARQUITETURA foi atualizado se o código
mudou. Se a mudança não afeta a arquitetura documentada, diga explicitamente
'ARQUITETURA sem alteração necessária'.
